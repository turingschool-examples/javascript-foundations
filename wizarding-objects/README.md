# Wizarding Objects

### Installation Steps

1. run `npm install`
2. Run your first test suite ```npm test test/pet-test.js```
3. Make the tests pass
4. Lather, rinse, repeat.

### Make the tests pass in the following sequence:

* test/pet-test.js
* test/wand-test.js
* test/wizard-test.js
